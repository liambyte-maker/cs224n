# CS224n Assignment 2: Neural Dependency Parsing

This assignment focuses on implementing a transition-based neural dependency parser.

Key components implemented include:

- SHIFT, LEFT-ARC, and RIGHT-ARC transitions
- PartialParse and dependency construction
- Minibatch parsing for efficient sentence processing
- Embedding lookup using pretrained word embeddings
- Feedforward neural network with ReLU and Dropout
- Cross-Entropy loss and Adam optimization

Through this assignment, I gained hands-on experience with dependency parsing, neural network training, and the integration of NLP parsing algorithms with deep learning models.
